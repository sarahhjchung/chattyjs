Chatbox

A Chatbox object represents the chatbox itself and its functionalities. A chatbox object is customizable by colour, theme, and position.

Type parameters

Name Default Type Description
colour 'grey' String The main colour for the chatbox. Takes an acceptable colour name or a hex value (ie. #000000).
theme 'light' String Light mode or dark mode for the chatbox. Takes 'light' or 'dark' only.
right '20px' String The value for the right position of the chatbox (ie. moves right from the right).
left 'auto' String The value for the left position of the chatbox (ie. moves left from the left).

Constructor

To create a Chatbox object, you must call its constructor.

                    
    new Chatbox(colour, theme, right, left)

Functions

The following are functions you can call by accessing the Chatbox object.

Name Arguments Behaviours Return Value
makeChatbox none Generates a button at the bottom of the screen in the correct right and left position given in the constructor. Clicking this button will open the chatbox. void
minimizeChatbox none Minimizes the chatbox. void
openChatbox none Opens the chatbox. void
createInlineButton nodeElement: HTMLElement
width: String		 Adds an inline button at the end of nodeElement of size width. When clicked, user can open and minimize the chatbox. void
addChatbubble text: String
isUser: Boolean
hasLink: Boolean
href: String
hasImage: Boolean
src: String		 Adds a text message to the chatbox.

text is the string content of the message.
isUser determines whether the message is from the developer or user.
hasLink determines whether the message contains an interactive link.
href (default null) is the href that the link should lead to.
hasImage (default false) determines whether the message the contains an image.
src (default null) is the src of the image. 		void
addQuestion question: String
answers: String[]		 Creates a new Question object and stores the question in the chatbox list of questions. Returns the Question object. new Question
callQuestion questionID: Number Calls the question with questionID to show up in the chatbox. void
