About Chatty.js

Chatty.js is a library that provides developers with the UI for a generable chatbox to easily set up and use on their webapps. Chatty.js does not provide network communication but provides simple API calls for developers to set up their own flow of different questions and follow-ups to get the users more familiar with their company and product. Users will be able to type their responses to the questions in the chat and receive automatic responses based on how the developers set up the flow of conversation.

Popular Use Cases

Some popular use cases of this library include using the chatbox as a way to help users navigate the website and learn more about the company or product, direct and link users to helpful nooks and crannies in the site that might be hard to locate, and keep users engaged at all points on a large web app with appealing visuals.
