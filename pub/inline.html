Generate an Inline Chatbox

You can also add an inline chatbox if you wish! An inline chatbox button can be placed anywhere on your web app and will open/minimize the chatbox from there when a user clicks on it. This is a unique feature that might seem out of place, but can actually be useful for large web apps with a lot of content. It'll be a fun addition to help users be more engaged to each part of your web app.

Just like a full chatbox, after generating an inline chatbox, you will be able to add questions, links, images, and more to interact with your users and create a fun user experience!

Click on the inline icon on the right to see the example. 

                    
    const cb = new Chatbox("#E07A5F", "light")
    cb.makeChatbox()

    const q0 = cb.addQuestion("Do you like this new feature?", ["Yes", "No"])
    
    // Call the opening question
    cb.callQuestion(q0.questionID)
    
    // Create actions to follow q0 and link actions to question
    const a0 = new Action("Thanks!", null, false)
    const a1 = new Action("Aw that's alright", null, false)
    q0.getActions([a0, a1])

    // Get the element in your document you'd like to add the inline button next to
    const element = document.getElementById("exampleButton")
    cb.createInlineButton(element, "20px")
