Action

An Action object represents an action that will follow from a response to a question by a user. An action can be a simple text response, another question, or a combination of both.

Type parameters

Name Default Type Description
message none String The message in the chat bubble of this action.
nextQuestionID none Number The ID of the question that should be called after the user responds to the question that this action follows.
hasLink none Boolean The boolean that determines whether this action's response chat bubble contains an interactive link.
href null String The href that the link should lead to.
hasImage false Boolean The boolean that determines whether this action's response chat bubble contains an image.
src null String The src of the image.

Constructor

To create an Action object, you must call its constructor.

                    
    new Action(question, nextQuetionID, hasLink, href, hasImage, src)

Functions

The following are functions you can call by accessing the Action object.

Name Arguments Behaviours Return Value
evokeAction none Evokes the action.

If the action contains a message, it will send a chat bubble containing the message and necessary contents (ie. link, image). If the action has a valid nextQuestionID, it will then call the question of that ID. 		void
