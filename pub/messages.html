Simple Text Messages

You can set up an introductory message for users to see when they open up the chatbox for the first time using just addChatBubble.

Click on the icon on the bottom right of the screen to see the example (in dark mode theme). 

                    
    const cb = new Chatbox("#E07A5F", "dark")
    cb.makeChatbox()
    cb.addChatBubble("Welcome!")
    cb.addChatBubble("This is an example of a simple text message.")
