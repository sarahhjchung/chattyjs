Question

A Question object represents a question to be asked in the chatbox.

Type parameters

Name Default Type Description
question none String The question
answers none String[] The valid responses to the question.
actions [] String The actions with respect (by index) to the valid responses.
quetionID none Number The unique ID of this question.
formattedQuestion none String The formatted question that will show up inside the chatbox when called. Includes the valid responses at the end of the question.
formattedAnswers none String The message that includes the valid responses to prompt users who type in an invalid response to try again.

Constructor

To create a Question object, you must call its constructor.

                    
    new Question(question, answers)

Functions

The following are functions you can call by accessing the Question object.

Name Arguments Behaviours Return Value
getActions actions Assigns actions to this question's actions. void
onAnswer answer Checks that answer is a valid answer with leniency (ie. 'yes', 'YES' and 'yEs' are all valid for answer 'Yes'). Evokes the corresponding action to the valid answer. void
